# Pull Request Template

The canonical pull request template for this repository lives at
[`.github/pull_request_template.md`](../.github/pull_request_template.md).

GitHub auto-populates new pull requests from that file, so contributors do
not need to copy it by hand. Agents opening a PR (via `gh pr create` or
similar tooling) should read the canonical template at the link above and
fill in every section.
